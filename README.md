# はじめに
これは、Finatextグループのソフトウェアエンジニアの共通スキルテストです。

本スキルテストの内容・URLを第三者に共有することは禁止とします。

# スキルテストの概要
## 問題のカテゴリ
大きく、以下のカテゴリの問題で構成されています。

* Q1. Container
* Q2. JWT
* Q3. HTTP
* Q4. Network

## 問題の進め方
* 制限時間は72時間です。(具体的な締切日時は別途連絡いたします。)
* 期間中は、好きなタイミングで問題を進めて頂いて構いません。
* 困っていることや不明な点があれば、別途ご連絡する方法でご連絡ください。(なるべく素早くご返信しますが、時間帯によってはすぐのご連絡が難しい場合もあります。返信があまりにも遅れてしまった場合は、試験時間を延長させていただくこともあります。)
* 一部の問題では、インターネットから HTTP/TCP/80番ポート もしくは HTTPS/TCP/443番ポート にアクセス可能なサーバーをご準備いただく必要があります。

## 回答の方法
すべての問題で、以下のフォーマットで構成されたflagを取得することができます。
```
f1nat3xthd\{\[0-9a-f\-]{36}\} 例: f1nat3xthd{8c51221a-41a4-4e35-80d4-5b7adc74a753}
```
取得できたflagを以下のGoogle Formから送信してください。

https://forms.gle/DKsAgSvse12ofDTo6

なお、全ての問題のflagを提出することが必須ではありません。
取得できた範囲のflagを全て提出してください。

Google Formの回答は正解・不正解が確認可能、かつ回答の修正は可能になっているので、意図せず不正解になっている問題があれば修正してもらっても問題ありません。
(ただし、Google Formによるflagの総当りは禁止とします。)

## 面接による解法のレビュー
Google Formによる回答の提出後、スキルテストの結果の判断を行います。
通過した場合、次の選考プロセス(面接)において、各問題の解答方法について質問させていただくことがあります。
そのため、使用したコードは保管し、面接の際に画面共有等できる状態にしておいてください。
